# LuxeScribe Pro - Stationery Shop Management System

A modern AI-powered stationery shop management system built with React, Vite, Firebase, and Gemini AI.

## Features

- Google Authentication (Firebase Auth)
- Inventory Management
- Product Catalog
- Sales Tracking
- Customer Management
- AI Assistant Integration
- Firestore Database
- Responsive Dashboard
- Vercel Deployment

## Tech Stack

- React
- TypeScript
- Vite
- Firebase Authentication
- Cloud Firestore
- Google Gemini AI
- Tailwind CSS
- Vercel

## Environment Variables

Create a `.env` file and add:

```env
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=stationery-pro.firebaseapp.com
VITE_FIREBASE_PROJECT_ID=stationery-pro
VITE_FIREBASE_STORAGE_BUCKET=stationery-pro.firebasestorage.app
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
VITE_FIREBASE_FIRESTORE_DATABASE_ID=(default)

GEMINI_API_KEY=your_gemini_api_key
```

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Deploy easily on Vercel:

```bash
npm run build
```

## Firebase Setup

1. Create Firebase Project
2. Enable Google Authentication
3. Create Firestore Database
4. Add Authorized Domains
5. Configure Environment Variables
